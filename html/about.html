Tentang Today's Web

Today's Web merupakan website yang berguna untuk menampilkan informasi mengenai prakiraan cuaca dan info gempa bumi terbaru.

Website ini dibangun untuk memenuhi tugas akhir pelatihan WEB I Program Beasiswa PUB Angkatan 20

Dengan adanya web ini diharapkan dapat digunakan oleh pengguna yang akan beraktivitas diluar agar bisa lebih siap sebelum melalui hari-harinya.

Semua informasi yang ada didalam website ini menggunakan sumber dari openweather.com dengan akurasi data sekitar 80-85%
